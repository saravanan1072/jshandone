Q.8 Create examples to explain promise.all function

ANSWER: 

        let promise=new Promise((resolve,reject)=>{
            let count=10;
            if(count===10){
                resolve(count);
            }
            else{
                reject()
            }
           
           })
           promise
           .then((count)=>{
            console.log("promises resolved");
           })
           .catch(()=>{
            console.log("promises rejected");
           })
           .finally(()=>{
            console.log("end of promises");
           })
THEN -> if a promise is resolved then it enter into a then function. CATCH -> if a promise is rejected then it enter into a catch function. FINALLY-> if a promise is resolved or rejected then it enter into a catch function
